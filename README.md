# Система онлайн-записи - Справочники

Веб-интерфейс для управления справочными данными (Клиенты, Услуги, Специалисты) с CRUD-операциями, пагинацией, поиском и валидацией.

## 📋 Функционал

- ✅ Список записей с пагинацией (10 записей на страницу)
- ✅ Добавление новых записей с валидацией
- ✅ Редактирование существующих записей
- ✅ Удаление с проверкой связанных записей
- ✅ Поиск по текстовым полям
- ✅ Сортировка по любому столбцу
- ✅ Flash-сообщения об успехе/ошибке
- ✅ Страница детального просмотра (view.php)
- ✅ CSRF-защита форм
- ✅ Адаптивная вёрстка (мобильные устройства)
